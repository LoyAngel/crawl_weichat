## 📖 使用说明

1. **启动应用** - 运行 `npm run dev` 启动开发服务器
2. **输入链接** - 在输入框中粘贴微信公众号文章链接
3. **开始爬取** - 点击"开始爬取"按钮
4. **查看进度** - 实时查看下载和压缩进度
5. **下载文件** - 任务完成后下载压缩的图片文件
